# C++ Programming Tutorial

Comprehensive C++ tutorial for building a CLI-based Tic-Tac-Toe game.

## Folder Structure

1. **01_Basic_Syntax** - Hello World, Comments, Input/Output
2. **02_Variables_DataTypes** - Types, Operators, Constants
3. **03_Arrays_2D_Arrays** - Arrays, 2D Arrays, Game Board
4. **04_Loops** - for, while, do-while, nested loops
5. **05_Conditionals** - if-else, switch, logical operators
6. **06_Functions** - Parameters, return values, overloading
7. **07_Structs** - Data grouping, nested structs
8. **08_Classes** - OOP, constructors, encapsulation
9. **09_Enumerations** - Named constants, game states
10. **10_Pointers** - Memory addresses, references

## Compilation

```bash
g++ filename.cpp -o output
./output
```

## Essential Topics for Tic-Tac-Toe

- 2D Arrays (game board)
- Loops (game flow)
- Conditionals (move validation, win detection)
- Functions (code organization)
- Structs/Classes (player, game state)
- Enums (game status)
